# dio-sales-report
Sales Report project developed as part of a DIO challenge. The goal is to reproduce and organize a sales analysis project using data modeling and SQL queries. The repository contains database scripts, project documentation and supporting files, demonstrating practical data analysis and GitHub organization skills.

# Sales Report

Projeto desenvolvido como parte de um desafio da DIO (Digital Innovation One). O objetivo é reproduzir e organizar um projeto de análise de vendas utilizando modelagem de dados e consultas SQL, demonstrando na prática conceitos fundamentais de banco de dados e análise de dados.

Este repositório foi criado com foco em organização, documentação e boas práticas de versionamento utilizando Git e GitHub, permitindo que o projeto também funcione como item de portfólio técnico.

---

# Objetivo do Projeto

O objetivo deste projeto é construir uma estrutura de banco de dados e realizar consultas SQL que permitam analisar informações de vendas, gerando relatórios úteis para tomada de decisão.

Durante o desenvolvimento são aplicados conceitos como:

- Modelagem de dados
- Criação de tabelas
- Relacionamentos entre entidades
- Inserção de dados
- Consultas SQL para análise de vendas

---

# Tecnologias Utilizadas

- SQL
- MySQL / PostgreSQL
- Git
- GitHub
